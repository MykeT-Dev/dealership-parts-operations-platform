# Dealership Parts Operations Platform

A production-style full-stack application for managing dealership parts inventory, with separate public and admin views.

## Overview

This project is based on real-world dealership operations and uses sanitized inventory data to model realistic workflows.

## Features (Planned)

- Public parts browsing and search
- Supplier-based inventory management
- Admin dashboard for operations
- Cost and margin tracking (internal only)
- Inventory adjustment history

## Tech Stack

- Frontend: React + TypeScript
- Backend: FastAPI
- Database: PostgreSQL
